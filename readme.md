### Rentx

- Projeto desenvolvido no modulo React Native do Ignite da Rocketseat

### Instalação

- Comandos

```shell
# Instalar dependências
$ yarn

# Iniciar api
$ cd ./api
$ yarn
$ yarn start

# Iniciar projeto
$ yarn expo start
```
